What's it do?

easy video embed is a jQuery plugin that allows you to easily and dynamically embed YouTube and Vimeo videos onto a page. Play with the example to the rightbelow to see what it does.

Why use it?

easy video embed was inpired by a web app I was helping to create which had a news module. We allowed the user to create news articles with different types of media embeds, one of those being YouTube and Vimeo videos. All the users have to do is input a url and the video shows up. No copying embed code. No figuring out what the correct url to use is. Just simply copy & paste.
